

# Employee & Department Management System

## Project Overview
This project is a SQL-based database management system designed to manage employee and department records. It demonstrates relational database design, data management, and analytical querying using SQL.

## Objective
The main goal of this project is to:
- Store employee and department information
- Maintain relationships between tables
- Perform workforce analysis using SQL queries
- Generate business insights from employee data

## Tools Used
- :contentReference[oaicite:1]{index=1}
- SQL

## Database Design

### Tables Created

#### 1. Departments Table
Contains department information:
- Department ID
- Department Name

#### 2. Employees Table
Contains employee information:
- Employee ID
- Employee Name
- Age
- Department ID

## Concepts Implemented

### Database Design
- Created relational database schema
- Used Primary Keys
- Used Foreign Keys
- Maintained data integrity

### SQL Operations Performed
- SELECT statements
- WHERE conditions
- ORDER BY sorting
- GROUP BY analysis
- HAVING clause
- INNER JOIN
- Aggregate Functions
  - COUNT()
  - AVG()
  - MAX()
  - MIN()
- Subqueries
- CTEs
- Views

## Business Analysis Performed

Using SQL queries, this project analyzes:

- Employee distribution by department
- Average employee age by department
- Departments with multiple employees
- Employees older than average age
- Top senior employees
- Department-wise employee counts
